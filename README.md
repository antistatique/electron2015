# electron2015 Styleguide

### Install the styleguide

```bash
$ npm install
$ bower install
$ gulp
```

### Build the styleguide with minification

```bash
$ gulp --production
```

### Build only the minimum required for production

```bash
$ gulp build
```

### Watch and compile files as you go

```bash
$ gulp serve
```

### Publish to GH Pages

```bash
$ gulp deploy
```


> This is a Yeoman generated styleguide for electron2015.
